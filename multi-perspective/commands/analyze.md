# /analyze Command

Quick command to invoke multi-perspective analysis.

## Usage

```
/analyze [proposition]
```

## Examples

```
/analyze "AI will replace most white-collar jobs within 10 years"
/analyze "Agile methodology is superior to Waterfall"
/analyze "Remote work is more productive than office work"
/analyze "We should fully adopt microservices architecture"
```

## What Happens

1. Invokes the multi-perspective-analysis skill
2. Clarifies and confirms the proposition
3. Generates relevant expert panel (4-6 experts)
4. User selects analysis mode:
   - **Validation**: Find blindspots and counterarguments
   - **Comprehensive**: Get detailed perspective from each expert
   - **Debate**: Watch experts engage in structured dialogue
5. Delivers structured analysis
6. Offers optional save to AkashicRecords

## Natural Language Alternatives

You can also invoke this skill naturally:
- "Analyze this from multiple perspectives: [proposition]"
- "What do experts think about [topic]?"
- "Challenge my assumption that [proposition]"
- "Multi-angle analysis: [proposition]"
- "Validate using different expert perspectives: [proposition]"
